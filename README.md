# Crypten

Encryption and decryption with RSA

## Getting Started
```
run git clone https://github.com/dhiafadhil/crypten.git
```

### generate RSA
create yout file python to generate rsa
```
from cryptenv2 import generate

generate.PublicAndPrivateKey()
```

### Run on shell or terminal

```
python3 <yout file python>.py
```
